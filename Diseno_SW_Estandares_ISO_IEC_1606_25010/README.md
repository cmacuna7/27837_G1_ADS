Este directorio contiene la guía de diseño de software que especifica los estándares y mejores prácticas basadas en ISO/IEC 1606-25010 para el proyecto **Healthy+**.

- **ISO/IEC 25010:** Modelo de calidad del producto software (características de calidad)

## Características de Calidad Cubiertas

La guía aborda las siguientes características de calidad según ISO/IEC 25010:

- **Funcionalidad:** Completitud y corrección de las funciones
- **Fiabilidad:** Madurez, disponibilidad y tolerancia a fallos
- **Usabilidad:** Capacidad de comprensión, aprendizaje y operabilidad
- **Eficiencia del Desempeño:** Comportamiento temporal y utilización de recursos
- **Seguridad:** Confidencialidad, integridad y no-repudio
- **Compatibilidad:** Coexistencia e interoperabilidad
- **Mantenibilidad:** Modularidad, reutilizabilidad y analizabilidad
- **Portabilidad:** Adaptabilidad e instalabilidad
